Hi there,

this is our little christmas present for the world. We give you our framework "gwwFramework" which is supposed to help developers creating big websites. The following pages use the framework.

http://www.ctelly.com http://www.skelpo.com http://www.pray4me.eu and many more

We kept the documentation very simple so far and will extend it in the future. Feedback, corrections and fixes are always appreciated!

Keep in mind: This framework is actually supposed to help so please don't expect a formally and perfectly organized framework. It is probably quite a dirty programmed framework BUT - and that counts for us - it is super flexible, effective and powerful. But see for yourself - we love it. ;)
